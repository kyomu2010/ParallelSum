Sample console output for 100000000 items:

```
8
655326472
Sequential sum takes: 61ms
Sum is: 655326472
Parallel sum takes: 44ms
```

Sample console output for 1000 items:

```
8
49038
Sequential sum takes: 0ms
Sum is: 49038
Parallel sum takes: 3ms
```
